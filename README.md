# Authentication Project
A Node.js authentication system built using Express, Passport.js, and PostgreSQL.  
It includes features like user registration, login, session management, and OAuth (Google login).
Installation & Setup

Show how to run the project locally.

## Installation

1. Clone the repository  
   ```bash
   git clone https://github.com/2100090048/Authentication-Project.git
   cd Authentication-Project


Install dependencies

npm install


Create a .env file and add:

SESSION_SECRET=your_secret_key
GOOGLE_CLIENT_ID=your_client_id
GOOGLE_CLIENT_SECRET=your_client_secret


Start the server

node solution.js
